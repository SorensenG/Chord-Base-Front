# ChordBase Figma Board

Arquivo gerado:

- `chordbase-figma-board.svg`

Como abrir no Figma:

1. Abra o Figma.
2. Crie um arquivo novo.
3. Arraste `design/figma/chordbase-figma-board.svg` para dentro do canvas.
4. Use `Ungroup` para editar textos, cores, cards e frames.

Observacoes:

- O arquivo e um board editavel em SVG, nao um `.fig` nativo.
- O icone escolhido como direcao visual e `design/app-icons/chordbase-icon-c-acorde-sem-xo.png`.
- A estrutura cobre mobile iOS/Android e web com a mesma linguagem visual.
